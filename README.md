# RIZEK Trip Sorter

RIZEK Trip Sorter is a simple web app to help their users to find the cheapest
or the fastest way to travel from a city to another.

## Steps to run

> Install Packages

```
npm i
```

> For development purpose:

```
npm run start:dev
```

> To run as Docker container:

```
npm run start:docker
```

> To run unit tests:

```
npm run test
```
